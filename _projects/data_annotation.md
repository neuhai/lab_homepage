Data annotation represents a significant proportion of domain experts' daily work. Doctors give diagnoses based on lab test results, judges and jurors give judgments based on Facts and Law Articles, etc.,  all these works are some kind of data annotation. By using AI-assisted data annotation tools, domain experts can receive real-time updated suggestions while doing their daily work.
